# Web Public

Public marketing website for Everbloom platform.

## Status

This application is currently a placeholder and will be implemented in the future.

## Planned Tech Stack

- Next.js or Vite
- TailwindCSS
- React

## Structure

- `src/` - Source code (to be implemented)
- `public/` - Static assets
- `docker/` - Docker configuration files

Container image is built from repo root: `docker build -f docker/Dockerfile.web-public -t ghcr.io/everbloom/web-public:latest .`
